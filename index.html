🧺 収穫予定
ℹ️ この表について（凡例・畝の状況・果樹）

献立の目安。天候・生育で前後する見込み。定植日・株数から現実的に採れるものだけを載せる。 （多年）（宿根）（こぼれ種）（自生）＝植え直さなくても毎年ほぼ放任で出るもの（場所も決まっている）。印の無いものは毎年まき直す一年生。

畝が足りていない：この作付け（4人家族ぶん）に対し、畑は現在 計11畝（東9・西2／各0.7×2m）。晩秋ピークで約8畝不足（先行開墾が必要。にんにく斜面振替＋西の畝11増設で各約1畝ぶん軽減済み）。

果樹・きのこは今年はまだ：リンゴ・ウメ・ブルーベリー・キウイは2026年5月に植えた若木（活着済み）、シイタケは2026年6月の種駒で、初収穫は数年先。初収穫の目安は下の「長期の見通し」に年度で載せた。当面は野菜と自生資源が中心。（鉢植えのレモンは別で、11〜12月に色づいたものから採れる＝下記）

長期の見通し（活着済みの株・初収穫の目安）

一度根づけば毎年採れる木・宿根。すべて2026年に植えて活着済み。※品種・生育・剪定で前後する目安。まだ植えていないもの（さくらんぼ・もも・サルナシ・アスパラガス・ルバーブ・チャイブ等）は植えたら追記する。

  • 2027年〜：ミョウガ（本格化）・ニラ（安定）。シイタケが発生開始（2026-06 種駒／秋に初発生の可能性）
  • 2028年〜：ブルーベリー×2（シャープブルー・ジョージアジム）が少しずつ。シイタケ安定
  • 2029〜2031年：ウメ（コウメ）→ リンゴ（フジ・ツガル）→ キウイ（ヘイワード♀・受粉樹トムリ♂）が実り始める（若木の3〜5年目）

1月 ｜ 冬・農閑期
気にすること
  • 落葉果樹の剪定適期（芽が動く前）。晩霜はまだ先で、いまは計画と土づくりの時期
  • レモン鉢は室内で越冬中（水やり控えめ・過湿で根腐れ注意）
行動
  • 果樹の休眠剪定＋寒肥：リンゴ・キウイ・ウメ・ブルーベリー
  • 雪囲い。次シーズンの計画づくりと種苗の発注
  • 畑を広げる開墾・粗起こし（1〜2 月）。育苗設備の準備
詳しく（施肥・注意など）
  • 落葉果樹の休眠期管理（12〜2 月）：開花・展葉の前に剪定し、有機質の寒肥を入れる。
  • 冬の土づくり（寒起こし）：収穫後の空き畝を粗く掘り返して寒気に当て、越冬害虫・病原を減らす。半熟コンポストはここですき込む（これから植える苗の株元・根菜の播種直前には使わない）。
  • 資材のまとめ買い（中庸・純作付 約 47㎡ 想定）：完熟牛ふん堆肥 40L×8 袋、苦土石灰 20kg×1、化成 8-8-8 20kg×1、タネ一式。堆肥は約 117kg と物量が多いので配達／複数回運搬を見込む。
2月 ｜ 冬〜早春・準備
気にすること
  • 剪定の締切（芽が動く前まで）。春の自生資源（フキノトウ・ヨモギ）に備える月
  • 石灰は「定植の 2 週間以上前」に入れる ── 逆算して今から
行動
  • 剪定を終わらせる。果樹・宿根に寒肥（有機質）
  • 拡張地を含め苦土石灰を全面散布（pH 調整）
  • 春播種・春植えの準備。自生資源の収穫カレンダーを再確認
詳しく（施肥・注意など）
  • 寒肥＋石灰：果樹・宿根は寒肥（12〜2 月）。畑は苦土石灰を全面に（多用しない・100〜150g/㎡）。
  • 自生資源の再確認：毎年 2 月にフキノトウ・ヨモギ若芽の適期（月末〜3 月）を思い出し、見回りを始める準備。
  • 春の段取り：ジャガイモ等の春植え、ナス科の加温育苗（3 月）に向けた場所・資材の確保。
3月 ｜ 早春・始動
気にすること
  • ナス科は発芽適温 25〜30℃ ── 加温しないと育たない（室内＋育苗マット）
  • 晩霜の警戒開始（当地は 5 月まで遅霜の記録あり）
  • 自生資源が採れ始める。春を逃さない
行動
  • ナス科（トマト・ピーマン・パプリカ）を種から加温育苗開始
  • レタス播種（3〜5 月）。長ネギ（根深）を種から育苗開始（3 月下旬〜4 月）
  • イチゴに春の追肥・マルチ。石灰の 1 週間後に堆肥をすき込み
  • フキノトウ（〜4 月）・ヨモギ若芽（3〜5 月）・タラの芽（斜面・4〜5 月）の収穫。ツバキ（茶用・同定でき次第）
詳しく（施肥・注意など）
  • レタス：好光性なので覆土は 2〜3mm 以下、発芽まで土表面を湿潤に、半日陰で管理。
  • ヨモギの同定：葉裏が白く綿毛・揉むと香る株のみ。トリカブト（葉裏白くない・無臭・猛毒）と誤認しない。草餅用は重曹で下茹で→アク抜き。
  • 宿根資源の春植え：アスパラ大苗・ミツバ・ウド。ミツバ・ウドは半日陰・湿った場所（フキと同じ環境）へ。
  • タラの芽：斜面自生。頂芽が開く直前が食べごろ→天ぷら。1 本から頂芽を採り尽くすと木が枯れるので、側芽/2 番芽は残す。幹に鋭い棘。
  • ツバキ（茶用）：斜面自生。ヤブツバキなら冬〜早春の花で椿茶、チャノキなら春の新芽で緑茶。用途・適期が変わるので冬〜早春の開花で同定してから。
  • 果樹の春植え適期入り（3 月中旬〜4 月）：検討中の桃・さくらんぼはここ。品種・受粉樹・耐寒性を購入前に一次情報で確認。
4月 ｜ 春・育苗と芽吹き
気にすること
  • キュウリは育苗 30〜40 日と短い ── 5 月定植に合わせて今まく
  • 宿根（ニラ・ミョウガ・シイタケ）の萌芽管理が始まる
行動
  • キュウリ育苗開始。アオジソ播種（4〜5 月）
  • ニラの萌芽・追肥、ミョウガの萌芽・敷き藁、シイタケの春発生・散水
  • フキ葉柄（4〜6 月）の収穫
詳しく（施肥・注意など）
  • 多年草の立ち上がり：ニラは初年は控えめに。ミョウガは敷き藁で乾燥防止。シイタケのホダ木は散水で春発生を促す。
  • 自生資源：フキ葉柄は塩で板ずり→茹で→水にさらし皮むき（アク抜き必須）。
5月 ｜ 春〜初夏・定植ラッシュ
気にすること
  • 遅霜明け（5 月中〜下旬）が解禁ライン ── 果菜の定植とレモン搬出はここまで待つのが安全
  • マメ科は元肥半量（多いと「つるボケ」）
行動
  • トマト・ピーマン・パプリカ・キュウリ・サトイモ・ネギを定植。支柱・防虫ネット
  • エダマメ・インゲンを直播。ナガイモのつる更新・植付
  • レモン鉢を屋外へ搬出（5 月中〜下旬。数日は半日陰で慣らす）
  • 自生：タラの芽の残り、ウワミズザクラの蕾＝アンニンゴ（斜面・要同定／湯通し→塩漬け）。花期なので同定もここで
詳しく（施肥・注意など）
  • 遅霜の前科：2026 はリンゴ「ツガル」が晩霜被害。GW 明けより下旬まで待つのが安全。
  • ウワミズザクラ（アンニンゴ）：斜面自生・未同定（イヌザクラと酷似→花穂の柄の基部に葉があるのがウワミズザクラ）。蕾～若い青実を塩漬け（杏仁香）。核・未熟部に青酸配糖体→蕾は湯通し・青実は塩漬けで処理必須。色づく前の緑を摘む（窓が短い）。
  • レモン搬出：いきなり直射に出さず慣らす。以降は屋外で生育。
  • レタス：5 月下旬〜6 月は日中 25℃ を超えやすく高温休眠。朝の涼しい時間にまき、発芽まで涼しく。
6月 ｜ 初夏・追肥と収穫始まり
気にすること

  • 長ネギ（根深）の溝植え定植（6〜7 月）。梅雨入りで病気管理
行動

  • 長ネギを溝植え（60 本）→ 以降、土寄せを繰り返す
  • イチゴ（5〜6 月）の収穫。※ブルーベリー・ウメ等は2026年植えの若木で、収穫は数年先（管理のみ）
  • 自生：ウワミズザクラの青い未熟果＝アンニンゴ（斜面・色づく前の緑のうちに・塩漬け）
詳しく（施肥・注意など）

  • 収穫：ブルーベリー・ウメ等の落葉果樹は 6〜7 月が本来の目安だが、2026年植えの若木は数年先。導入検討中のさくらんぼも同様。
7月 ｜ 盛夏・収穫最盛＋秋冬の仕込み開始
気にすること
  • 当地は秋が早い ── 暖地の目安より 1〜2 週早める
  • 梅雨×高温で種が流れる・徒長する ── ポット育苗（軒下）を優先
  • 秋冬野菜の播種本番に入る
行動
  • 秋冬野菜の育苗・直播：キャベツ・ブロッコリー育苗、ニンジン直まき、コマツナ・ハツカダイコン・カブのずらしまき
  • まいた日から防虫ネット。果菜は追肥・収穫最盛
  • キュウリのずらし定植：育苗中の後続苗を本葉2〜3枚で順次定植（初期株が病気で衰える前に後続を用意＝収穫を切らさない）。誘引・整枝・べと病/うどんこ病の防除も継続
  • 斜面の種球を手配（ラッキョウ・ワケギ・寒地系ニンニク）
  • 敷地の下草刈り（草が伸びる盛り）。刈った草・枝葉は炭素資材/マルチに回す
  • イチゴのランナー受け（7〜8月）：伸びた子株をポットで受けUピンで固定。次郎・三郎苗を使う（太郎苗は避ける）。9月まで養成→9〜10月定植。来季の株更新・増収の要
詳しく（施肥・注意など）
  • 梅雨×高温の実務：直まきよりポット育苗優先（種流亡・徒長・種腐れを避ける）。アブラナ科は 8 月がヨトウ・アオムシ最盛期→防虫ネット前提。
  • 下草刈り：初夏〜夏（6〜8 月）が本番。樹木の間伐は冬（12 月〜2 月）に分ける。刈草・枝葉はコンポストの炭素材や敷きマルチに使える。
  • ニンジン：好光性で覆土薄く、雷雨で流れるので不織布/板で覆う。晩抽性品種。
  • 種球の締切：ラッキョウ・ワケギは 8 月下旬〜9 月植付、ニンニクは 9 月上〜中旬植付 ── 種球は 7 月末〜8 月に手配。

仕組み（何をやっているか）

一言でいうと「畑の記録を Markdown のテキストで貯め、AI に読み書きさせて、判断と公開を自動化する」仕組み。核は3つ：

  • 記録: 日々の観察・作業を、決まった形のテキスト（Markdown）で1つのリポジトリに貯める。写真より「事実の言語化」を優先する。
  • 分析: 気象の実測値を貯め、積算温度（GDD）で「夏まきでも収穫に間に合うか」等を数値で逆算する。失敗は原因を実測で切り分け、来年の「着手条件」に翻訳する。
  • 公開: 記録のうち無害な一部だけを、このような1枚の静的 HTML に自動生成して家族へ共有する。個人や場所が特定される情報は、自動チェックで機械的に除外する。

AI（Claude Code 等のコーディング支援 AI）に「作業手順書」を1ファイル渡しておくと、写真やメモを送るだけで、AI が対象を特定し、記録を追記し、分析し、公開ページまで更新してくれる。人は現場で観察して伝えるだけでよい。

用意するもの
  1. 記録用のリポジトリ（git）: テキストを貯める場所。非公開で良い。GitHub の private リポジトリが手軽。
  2. AI アシスタント: Claude Code など、ファイルを読み書きできるコーディング支援 AI。手順書に沿ってテキストを更新させる。
  3. 手順書（AI への指示ファイル）: リポジトリ直下に、記録・分析・公開の手順を1ファイルで書いておく（Claude Code なら CLAUDE.md）。AI は毎回これを読んで同じ流れで動く。

手順書には最低限これを書いておくと、AI が迷わず回せる：

  • 記録の置き場所と書式（下の「日々のログ」）
  • 気象データの取得元と貯め方（下の「データと分析」）
  • 公開ページに「出してよいもの／絶対に出さないもの」の線引き（下の「公開の作られ方」）
日々のログ

  • 判断: 初収穫は数日内。黄化は生理的な下葉落ちで問題なしと見る。

  • 次: 週明けに収穫。追肥は見送り。 ```

  • 語彙のブレを防ぐため、作物名の一覧と場所名の一覧を別ファイルに持ち、ログはその表記に揃える（AI に「一覧に無ければ追記してから使って」と指示しておく）。

  • 「どの株をどこに植えたか」の台帳、「どの畝に何を作るか」の作付け表（連作を避けるチェック付き）も別ファイルで持つと、AI が現状を把握して次の一手を出せる。
データと分析

体感でなく実測で判断できるように、気象データを貯めて数値モデルで逆算する。

  • 実測を貯める: 最寄りの気象観測点の日別データ（平均気温・日照時間・降水量）を、AI に定期的に取得させて thermal-data.tsv のような表に追記する。
  • 積算温度（GDD）で逆算する: 「定植から収穫まで」に必要な積算温度を、過去の記録から較正しておく。すると「今から播いて収穫に間に合うか」「収穫はいつ頃終わるか」を数値で答えられる。小さな計算スクリプトを1本持たせておくと AI が呼び出して使う。
  • 失敗を来年へ渡す: 徒長・不発芽・生育不良などが出たら、原因を実測（気温・日照・降水）で切り分け、「来年はこの条件を満たしたら着手する」という月日でなく条件の形で1件ずつ書き残す。翌年の判断精度がそのぶん上がる。

要は「観察 → 記録 → 実測で検証 → 来年の着手条件」というループを、AI に回させ続ける。土地固有の年間カレンダーは外部に売っていないので、自分で1〜2年積むしかない。

公開の作られ方（安全に共有する）

このダッシュボードは、記録リポジトリから無害な数ファイルだけを取り出し、1枚の静的 HTML に自動生成したもの。安全のために二重の仕組みを入れている：

  • 許可リスト（allowlist）: 公開ページに載せてよいファイルをあらかじめ列挙し、それ以外は載せられないようにする。
  • 禁止パターン検査（denylist）: 生成した HTML を機械的に走査し、個人や場所が特定される情報・住まいや設備の情報・通信の情報などが1つでも混じっていたら、発行を止める。
  • 検索避け（noindex）を付け、公開リポジトリには履歴を残さず単一コミットで置き換える（ぼかす前の版が履歴に残らないようにするため）。

この「載せてよいものだけを、機械チェックを通してから出す」線引きを AI の手順書に書いておけば、うっかり機微情報を公開する事故を構造的に防げる。

作りの詳細まで踏み込みたい場合は、上のプロンプトのあとに「手順書（指示ファイル）のたたき台を書いて」「気象データ取得と積算温度の計算スクリプトの雛形が欲しい」と続ければ、AI が具体的なファイルまで用意してくれる。
